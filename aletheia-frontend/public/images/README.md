# Images Directory

Place the Aletheia background image here as `aletheia-background.jpg` (or `.png`).

The image should be:
- High resolution (recommended: 1920x1080 or larger)
- The vintage book with mirror and eye design
- Will be displayed as a muted background (15% opacity) with a dark overlay

If using a different filename, update the path in `app/page.tsx`:
```tsx
backgroundImage: 'url(/images/your-image-name.jpg)',
```
